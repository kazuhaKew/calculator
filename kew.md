# Enhanced Age Calculator

A modern, feature-rich age calculator web application that provides comprehensive age-related calculations and insights.

## Features

### Core Functionality
- Calculate exact age in years, months, and days
- Real-time age calculation as you type
- Support for different date formats
- Responsive design for all devices

### Enhanced Features
- **Detailed Breakdown**: Shows age in multiple units (years, months, weeks, days, hours, minutes, seconds)
- **Next Birthday Countdown**: Displays time remaining until next birthday
- **Zodiac Sign**: Automatically determines and displays zodiac sign
- **Life Statistics**: Fun facts like total days lived, heartbeats, breaths taken
- **Historical Context**: Shows what was happening when you were born
- **Age Comparison**: Compare ages with famous people or historical figures
- **Leap Year Detection**: Identifies if birth year was a leap year
- **Day of Week**: Shows what day of the week you were born

### Technical Features
- Input validation and error handling
- Local storage to remember last calculation
- Print/share functionality
- Dark/light theme toggle
- Internationalization support
- Accessibility compliant (WCAG 2.1)

## Technology Stack
- HTML5 with semantic markup
- CSS3 with modern features (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript (ES6+)
- Progressive Web App capabilities
- Responsive design principles

## Usage
1. Enter your birth date using the date picker
2. View comprehensive age calculations instantly
3. Explore additional insights and fun facts
4. Share or print your age report
5. Toggle between light and dark themes
